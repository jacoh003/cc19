THINGS I LEARNED:

1. You can use pseudo-class selectors to make more specific selections within a class without having to create another class in the html file.

2. If some CSS isn't applying to an element you are trying to select, see if there is a more specific selection that is being made in the CSS. The more specific selection will overrule the more general one.

3. You can see what kind of box model is being used by looking at the DevTools in your browser.

4. "Normal Flow" is already built into HTML. By default, this layout prevents things from overlapping because the content is aware of each other. Using the right HTML tags will allow you to make the most use of this built-in "Normal Flow."

5. You can animate on HTML, however you can do even more with animation if you involve JavaScript, SVG, or Web Animation API.
